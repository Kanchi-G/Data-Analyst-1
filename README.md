Handwritten Digit Recognition (MNIST)
Project Overview
This project builds a machine learning model to classify handwritten digits using the MNIST dataset.

Technologies Used
Python
NumPy
Scikit-learn
Matplotlib
Project Steps
Load MNIST dataset
Data preprocessing and normalization
Train machine learning model
Evaluate model performance
Results
The model classifies handwritten digits and evaluates performance using accuracy score and confusion matrix.
